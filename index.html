How does the auth middleware work?
AI
The auth middleware in middleware/auth.ts validates JWT tokens on every request. It:

1. Extracts the Bearer token from the Authorization header
2. Verifies it against the secret using jsonwebtoken
3. Attaches the decoded user to req.user
